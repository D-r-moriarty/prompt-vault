# 提示词保管箱 - 设计规格

## 概述

一款专为 AI 提示词管理设计的工具，采用**"灵感工坊"**概念——让创作提示词像在精心设计的工坊中工作一样愉悦。

**目标用户**：需要管理大量 AI 提示词的用户（设计师、开发者、创作者）

---

## 美学方向：Craft Workshop（工匠工坊）

**核心理念**：提示词是创作者的工具，就像木匠的工具一样需要精心整理、归类、取用方便。

**视觉风格**：
- 温暖的暖色调（米色、琥珀色、深棕色）
- 纸质纹理背景
- 手写体/书法风格的装饰元素
- 卡片式布局，模拟便签/文件夹
- 微妙的光影效果（模拟真实纸张层次）

**色彩系统**：
```css
:root {
    /* 主色调 - 暖琥珀 */
    --amber-50: #FFFBEB;
    --amber-100: #FEF3C7;
    --amber-200: #FDE68A;
    --amber-300: #FCD34D;
    --amber-500: #F59E0B;
    --amber-600: #D97706;
    --amber-700: #B45309;

    /* 背景 */
    --paper: #FAF7F2;
    --paper-dark: #F5F0E8;
    --ink: #1C1917;
    --ink-muted: #57534E;

    /* 功能色 */
    --sage: #84CC16;        /* 成功/已同步 */
    --terracotta: #EA580C;  /* 提醒/AI */
    --slate: #64748B;       /* 次要操作 */
}
```

**字体**：
- 标题：`"Noto Serif SC", "Source Han Serif CN", serif`（中文宋体，有文化感）
- 正文：`"Noto Sans SC", "Source Han Sans CN", sans-serif`
- 标签/数字：`"JetBrains Mono", monospace`（等宽，清晰）

## 功能模块

### 1. 分类系统（增强版）

**多级分类**：
- 一级分类：文件夹式（如"写作"、"代码"）
- 二级标签：更细粒度（如"文案"、"翻译"、"代码审查"）
- 支持拖拽调整顺序

**智能分类建议**：
- AI 分析提示词内容，自动建议分类
- 用户确认后可自动归类

**界面**：
```
侧边栏分类：
├── 📁 写作 (23)
│   ├── 📝 文案 (12)
│   ├── 📝 翻译 (8)
│   └── 📝 邮件 (3)
├── 📁 代码 (45)
│   ├── 🔧 审查 (20)
│   ├── 🔧 重构 (15)
│   └── 🔧 测试 (10)
├── 📁 对话 (34)
├── 📁 分析 (28)
└── 📁 未分类 (6)
```

### 2. 导入/导出功能

**导入**：
- 支持 JSON 格式（标准格式）
- 支持 Markdown 格式（每条用 `---` 分隔）
- 支持 CSV 格式（标题,内容,分类）
- 拖拽上传
- 导入时显示预览，允许编辑映射

**导出**：
- 全部导出为 JSON
- 按分类导出
- 导出为 Markdown（便于阅读）
- 导出为 Prompt 市场格式（可分享）

**界面**：
```
导入对话框：
┌────────────────────────────────────────┐
```

- 编辑中：表单模式

## 交互细节

### 动画效果

**页面加载**：
- 侧边栏先滑入（200ms）
- 卡片网格依次浮现（stagger 50ms）
- 标题淡入

**卡片交互**：
- 悬停：translateY(-2px), box-shadow 加深
- 点击：scale(0.98) 反馈
- 展开：slideDown + fadeIn

**AI 处理**：
- 思考动画：三个点依次出现
- 完成：绿色勾号 + 轻微震动

### 快捷操作

- `Cmd/Ctrl + N`：新建提示词
- `Cmd/Ctrl + F`：聚焦搜索
- `Cmd/Ctrl + I`：导入
- `Cmd/Ctrl + E`：导出
- `Cmd/Ctrl + K`：AI 整理

### 通知系统

- 右下角 toast
- 3秒自动消失
- 成功：绿色 + ✓ 图标
- 错误：红色 + ✗ 图标
- AI 处理：琥珀色 + ⚡ 图标

## 设计亮点

1. **纸质纹理背景**：细微的噪点纹理，模拟真实纸张质感
2. **便签式卡片**：轻微的阴影和旋转角度，模拟贴在墙上的便签
3. **墨水色调**：用深棕色代替纯黑，更温和有文化感
4. **温度计图标**：显示同步状态，有趣且直观
5. **AI 处理动画**：使用墨水晕染效果，暗示"思考"过程
